# MVP: Resumen Ejecutivo

## 🎯 En 1 Línea

**App que calcula automáticamente tus impuestos y te muestra cómo ahorrar legalmente.**

---

## ✅ Qué SÍ Incluir (MVP)

### 1. Calculadora de Ganancias
- Ingresás tus datos → Te dice cuánto pagar
- Considera deducciones automáticamente
- Resultado claro: "Debés $X" o "No alcanzás el mínimo"

### 2. Calculadora de Monotributo
- Calcula tu cuota mensual
- Alerta de recategorización: "Te quedan $X para facturar"
- Compara Monotributo vs Responsable Inscripto

### 3. Optimizador de Deducciones
- Lista de deducciones posibles
- Calcula ahorro: "Si deducís $X, ahorrás $Y"
- Sugerencias personalizadas

### 4. Dashboard Principal
- Resumen: "Este mes debés $X"
- Próximos vencimientos
- Oportunidades de ahorro

### 5. Recordatorios
- Notificaciones de vencimientos
- Calendario de pagos

### 6. Educación Básica
- "¿Qué es el Monotributo?"
- "¿Cómo funcionan las deducciones?"
- Explicaciones simples, sin jerga

---

## ❌ Qué NO Incluir (MVP)

- ❌ Facturación electrónica (muy saturado)
- ❌ Pagos directos (muy complejo)
- ❌ Integración multi-organismo completa (V2)
- ❌ IA avanzada (V2)
- ❌ Historial automático de pagos (V2)

---

## 🏗️ Stack Técnico Mínimo

### Frontend
- Web App (React/Vue) + PWA
- Responsive (mobile-first)

### Backend
- API REST (.NET Core / Node.js)
- PostgreSQL / SQL Server
- JWT para autenticación

### Integraciones
- AFIP API (solo lectura)
- Email (notificaciones)
